## Project Details

### Phase 1: SQL Database Management
- **Hotel Management System**: Designed database schema with proper relationships
- **Clinic Management System**: Created database structure for healthcare operations
- Implemented queries for data retrieval and analysis

### Phase 2: Spreadsheet Analysis
- **Ticket Analysis**: Analyzed support ticket data using Excel formulas and pivot tables
- File: `Spreadsheets/Ticket_Analysis.xlsx`
- Includes data visualization and summary statistics

### Phase 3: Python Data Processing
- **Time Converter** (`01_time_converter.py`): Converts time in minutes to human-readable format (HH:MM)
- **Duplicate Remover** (`02_remove_duplicates.py`): Removes duplicate characters while preserving order

## Assumptions & Approach

### SQL
- Used appropriate data types for optimal storage
- Implemented primary and foreign keys for data integrity
- Designed normalized schemas to avoid redundancy

### Python
- Basic Python implementation using loops and string manipulation
- Focused on clarity and efficiency
- Used basic data structures (strings, loops) as per skill level

### Spreadsheet
- Used Excel formulas for calculations
- Created pivot tables for summary analysis
- Applied conditional formatting for better visualization

## How to Use
